HTML Links: CheatSheet


Hyperlinks:

<a>

This anchor tag is used to create a hyperlink on an HTML document.

Type of Link:

Attribute:

Description:

Example:
Text links
<a href="">'link'</a>
This text becomes the recognised hyperlink to another page
<a href="http://www.google.com">Back to Google</a>
Internal links
<a href="#">'link'</a>
This will create a link to somewhere on the same page with the matching id
<a href="#contact">'Contact'</a>
Image Links
<a href=""> <img src=""/>'link'</a>
This create a link in an image placed on the page
<a href="http://www.google.com"> <img src="googlelogo.jpg"/> </a>
Email links
<a href="">'email'</a>
This string will connect an email to accound to the attribute
<a href="mailto:email@yourmail.com">'email@yourmail.com'</a>

Link Element:

<link>

This element is used to connect the HTML document to an outside resource, like CSS style sheets. This element is self-closing.

Type of Link:

Attribute:

Description:

Example:
<link/>
rel=""
This describes the format of the external source
<link rel="stylesheet" href="an/external/source.css"/>
href=""
This contains the filepath to the external source
